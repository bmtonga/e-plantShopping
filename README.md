# 🌿 e-PlantShopping Application

A modern web-based shopping application for browsing and purchasing household plants online. The platform allows users to explore available plant listings, add selected plants to a shopping cart, adjust quantities, remove items, and view the total purchase cost before checkout.

---

## 📌 Overview

**e-PlantShopping** is designed to provide a simple and user-friendly shopping experience for plant lovers. Users can browse plant categories, manage their cart, and keep track of total spending in real time.

---

## ✨ Features

- Browse available household plant collections
- View plant details including:
  - Name
  - Image
  - Price
  - Description
  - Sale tag 
  - Add plants to the shopping cart
  - Adjust item quantities directly from the cart
  - Remove items from the cart
  - Automatic calculation of:
   - Individual item totals
   - Grand total cost

---

## 🖥️ Application Pages

### 1. Landing Page
The landing page introduces users to the application.

#### Contents:
- Welcome message
- Brand tagline
- About section
- Call-to-action button

---

### 2. Product Listing Page
Displays all available household plants.

#### Contents:
- Products displayed in a responsive grid layout
- Products grouped by category
- Each product includes:
  - Plant image
  - Plant name
  - Price
  - Description
  - Sale badge/tag
  - **Add to Cart** button

---

### 3. Shopping Cart Page
Allows users to manage selected products.

#### Contents:
- List of added items
- Quantity adjustment controls
- Delete/remove item option
- Total cost for each item
- Grand total displayed at the top

---

## 🛒 User Flow

1. Open the application
2. Read the welcome information on the landing page
3. Navigate to the product listing page
4. Browse available plant categories
5. Add preferred plants to the cart
6. Open the shopping cart
7. Adjust quantities or remove items
8. View the updated grand total
